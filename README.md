# EasyGitOps

> Public WIP, not ready for production usage.

EasyGitOps is an opinionated CI+CD pattern for building and deploying services following simple best practices, and combining various open source tools in an effective manner.

## Prerequisites

- Go 1.22 or later

## Building

```bash
go build -o ego ./cmd/ego
```

## Usage

```bash
./ego
```
